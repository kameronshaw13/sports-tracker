# GameTracker Center v204

Focused follow-up to v203.

v203 fixed the horizontal centering but stacked the center-header items too high.
This version keeps the dynamic true-center horizontal anchor and restores the old
vertical lanes for:

- date
- time/final/final OT/final extra innings
- live inning label
- baseball diamond and count/outs
- odds/series/game labels

Only app/globals.css is changed.
